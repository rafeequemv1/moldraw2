What you get with MolDraw

  • 2D structure editing powered by Ketcher-style tooling in a focused MolDraw shell.
  • Integrated 3D visualization for geometry review and teaching.
  • Practical exports (for example PNG, SVG, and chemistry-friendly formats where supported).
  • Access to the same learning resources as the web app, including the course and FAQ.

System requirements (typical)

Windows 10 or later is recommended. The desktop build is a standard Windows installer; allow the installer to complete and start MolDraw from the Start menu or desktop shortcut if one is created. If SmartScreen appears, confirm you trust the publisher and source link you used (this page and moldraw.com).

Install steps

  1. Click Download Windows setup above.
  2. Run the downloaded installer and follow the prompts.
  3. Open MolDraw from the Start menu or desktop shortcut when the installer creates one.
  4. For the newest fixes without waiting for an installer refresh, keep using the web version.

Prefer not to install?

The browser app at moldraw.com is the reference experience and updates continuously. The Windows package is ideal when you want a desktop shortcut and a more app-like window.
