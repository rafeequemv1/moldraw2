Getting Started with MolDraw

This chapter helps new users start quickly with MolDraw. You will learn the 2D editor layout, structure search, and how to move from first sketch to 3D preview.

  • Open MolDraw and review toolbar + canvas areas.
  • Draw a basic structure and validate atom/bond placement.
  • Use search to load known molecules by name.
  • Switch to the 3D viewer and inspect rendering modes.
