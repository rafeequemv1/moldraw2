MolDraw AI

How to Use the AI Assistant

Draw, name, and explore molecules using Google Gemini — right inside MolDraw.

← Back to MolDraw

Step 1: Get a Free Gemini API Key

1

Go to Google AI Studio → API Keys

2

Sign in with your Google account.

3

Click "Create API key" and copy the key (starts with AIza...).

The Gemini API has a generous free tier — plenty for everyday use in MolDraw.

Step 2: Paste the Key in MolDraw

1

Open MolDraw and click the AI button (bottom-right of the 3D viewer).

2

Click the ⚙ gear icon in the chat header.

3

Paste your API key and click Save. It's stored locally in your browser — never sent anywhere except Google's API.

Step 3: Start Using It!

Type commands in the chat box. The AI can control the canvas and answer chemistry questions.

Draw molecules by name

You: draw aspirin
AI: Draws the aspirin structure on the canvas.

Name the current structure

You: what is this molecule?
AI: Tells you the IUPAC name and common name.

Show reactions

You: show Fischer esterification of acetic acid and ethanol
AI: Draws the reaction on the canvas.

Add structures without clearing

You: also add caffeine
AI: Adds caffeine alongside existing structures.

Ask about properties

You: what is the molecular weight of this?
AI: Calculates and tells you.

Clear the canvas

You: clear the canvas
AI: Clears everything.

Tips

  • The AI reads the current structure on the canvas (SMILES), so it always knows what you're working on.
  • Press Enter to send, Shift+Enter for a new line.
  • Your API key is saved in your browser's local storage. Clear it anytime via the ⚙ menu.
  • Powered by Gemini 2.5 Flash for fast responses.
